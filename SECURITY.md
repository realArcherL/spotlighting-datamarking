# Security Policy

## About This Package

`spotlighting-datamarking` is an **experimental** package implementing the [Spotlighting](https://arxiv.org/abs/2403.14720) technique from Microsoft Research. It is intended as a defense layer against indirect prompt injection — not as a standalone security guarantee.

## Scope

This package is experimental and comes with no warranty. The spotlighting technique reduces — but does not eliminate — the risk of indirect prompt injection. Always combine it with other security measures in production systems.
